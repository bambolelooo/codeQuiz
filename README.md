## About The Project

Code quiz site contains:

* Quiz of with four sample questions
* A timer that goes down by 5 seconds when the answer is not correct
* Locally stored scoreboard with sorting by score
* Cool wave svg from haikei

### Built With

* CSS
* HTML
* JavaScript
